# Party Game

A minimal mobile-first party game built with React, Vite, and TailwindCSS. It uses local JSON question files and requires no backend.

## Features

- Up to 4 saved player names
- Piccolo, Deep Talk, and Mixed game modes
- Local JSON questions with `{player1}` and `{player2}` placeholders
- Shuffled cards with immediate duplicate prevention
- Progress counter and touch-friendly controls

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

The generated `dist/` folder is ready for Vercel or GitHub Pages style static hosting.
